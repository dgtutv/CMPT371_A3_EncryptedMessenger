# SocketChat - Multi-Client Messaging App

Real-time chat using Python TCP sockets.

## How to Run
1. Terminal 1: `python server.py`
2. Other terminals: `python client.py`

Features:
- Broadcast messages
- Private messages (`/pm username message`)
- List users (`/users`)
- Quit with `/quit`

Limitations:
- No encryption
- No saved messages
- Single server
